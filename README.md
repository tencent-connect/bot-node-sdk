# QQ 频道机器人 SDK qq-guild-bot

用于开发 QQ 频道机器人的 Node.js SDK。

## 使用文档

[NodeSDK 文档](https://bot.q.qq.com/wiki/develop/nodesdk/)

## 本地开发

```shell
# clone repo
git clone https://github.com/tencent-connect/bot-node-sdk.git

# cd repo
cd bot-node-sdk

# run
npm run dev

# run example
npm run linkdev
npm run example
```
